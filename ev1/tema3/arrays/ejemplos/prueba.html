Métodos

  • El método array.push(valor) añade un nuevo elemento al array y nos devuelve su nueva longitud

    Utiliza los botones de arriba para ver el resultado⤴

  • El método array.concat(array) concatena los dos arrays

  • El método array.join(texto) devuelve una cadena de texto con los elementos del array separados por el texto indicado entre paréntesis

  • El método array.reverse() 🛑reescribe el array del revés

  • El método array.unshift(elemento) añade elementos al principio del array, devuelve la nueva longitud del array

    El método array.push(elemento) añade elementos al final del array, devuelve la nueva longitud del array ⏫Mirar más arriba para ver un ejemplo

    El método array.shift() elimina el primer elemento del array, devuelve el elemento eliminado

    El método array.pop() elimina el último elemento del array, devuelve el elemento eliminado

  • El método array.sort() ordena el array Alfabética o Numéricamente, también puede ser ascendente o descendente
